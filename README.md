## Project Name: Online Doctor

## Description:
This is a modern Next.js-based Online Doctor Appointment web application designed to provide patients with a fast, secure, and seamless platform to find doctors and book medical appointments online. The system allows users to browse a wide range of doctors by specialty, view detailed profiles including experience, hospital, availability, consultation fees, and ratings, and easily schedule appointments with just a few clicks. Built with Next.js for high performance, optimized rendering, SEO-friendly structure, and smooth user experience, the platform ensures full responsiveness across desktop, tablet, and mobile devices. It also includes essential features such as user authentication, appointment management, search functionality, and profile handling, enabling patients to update or cancel bookings efficiently. The application combines clean UI design with practical healthcare functionality, making it easier for users to access medical services anytime while improving the digital healthcare experience.

## Technologies Used.
Nextjs, 
Tailwind CSS, 
DaisyUI JavaScript (ES6+), 
React-Toastify (NPM Package), 
Backend System ,
JWT,
SwiperJS,
Better Auth,
mongoDB

## Features.
# Built with Next.js for fast, SEO-friendly performance
# All Doctors page to browse complete doctor listings
# Doctor details page with full profile information
# Easy appointment booking system from doctor profile page
# User dashboard to view all booked appointments
# Edit and update existing appointments
# Delete appointments when needed
# Secure login system for user authentication
# Only logged-in users can access doctors, booking, and profile
# JWT-based authentication for secure user verification
# Protected routes to prevent unauthorized access
# Fully responsive design for all devices (mobile, tablet, desktop)
